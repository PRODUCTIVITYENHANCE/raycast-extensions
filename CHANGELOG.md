## [1.0.0] - {PR_MERGE_DATE}

### Added

- **Add Md to Folder** command - Quickly save clipboard content to default folder
- **Add Md (Select Folder)** command - Save with subfolder selection and custom filename
- **Browse Markdown Files** command - Browse and search all Markdown files
- **Append to Markdown** command - Append content to existing Markdown files
- Multi-level subfolder support (up to 3 levels deep)
- Auto-handle duplicate filenames
- Default subfolder preference
- Custom editor preference (VS Code, Obsidian, Typora, etc.)
- Save and open in editor shortcut (Cmd + Shift + Enter)
